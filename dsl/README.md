# DSL Retrospektywy

Minimalny Process Pack:

```text
case-study/
├── experiment.yaml
├── authority.contract.aql
├── workflow.oql
├── pressure.eql
├── readiness.testql
├── situation.doql.json
├── events.sodl.jsonl
└── retrospective.md
```

`experiment.yaml` jest przenośnym kontraktem dla książki. Docelowo może być
wiązany z DSL-ami Subactora bez wpisywania nazw providerów do strategii.
